An initial value of the runtime. This is the empty.

eval :: MaruEnv -> SExpr -> IO (Either SomeException (SExpr, MaruEnv, SimplificationSteps)) Source #

Evaluate a S expression, and happen its side effects.

If you don't have a value of MaruEnv, you can use initialEnv.

Return an evaluated result, with new MaruEnv (env is changed if the evaluation of SExpr changes MaruEnv).
